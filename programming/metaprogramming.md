## Overview

**Metaprogramming** refers to a programming technique in which computer programs have the ability to treat other programs as data.

%%ANKI
Basic
What is metaprogramming?
Back: A technique for treating computer programs as data.

## Quotation

A **quotation** function is a function that converts source code into data.

%%ANKI
Basic
